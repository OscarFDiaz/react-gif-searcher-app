## Run Locally

Clonar el repositorio

```bash
  git clone https://github.com/OscarFDiaz/react-gif-searcher-app
```

Ir al directorio del proyecto

```bash
  cd my-project
```

Instalar dependencias

```bash
  npm install
```

Iniciar el servidor

```bash
  yarn dev
```
